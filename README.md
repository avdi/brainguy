# Brainguy

Brainguy is an Observer library. This is probably funnier if you are a 
Mystery Science Theater 3000.

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'brainguy'
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install brainguy
